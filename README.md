## Trello CloneCoding

### 1. 개발환경

#### backend

- platform : node js (14.17.3)

- framework : express (4.17.1)

- template engine : ejs (3.1.6)

#### frontend

- ECMAScript

### 2. 설치

#### /backend

- npm install

### 3. 실행

#### /backend

- npm run start
- http://localhost:3000 접속
